The data information about video games released between the years
2004-2010 can be found here. This data set is provided by CORGIS Dataset Project.

Some of the data statics that can be found in this data set are a
video games release date, the average time a player takes to complete
the main story of a game, ratings, the specific console for the game, and many more.
